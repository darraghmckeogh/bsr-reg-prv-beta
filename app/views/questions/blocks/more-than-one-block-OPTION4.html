Is {{ data['building-name'] }} a simple high-rise building?

A simple high-rise building might be a single standalone block of flats.

High-rise structures connected together

Buildings made up of multiple high-rise structures containing residential units are not simple high-rise buildings.

If your building contains multiple connected structures, but:

  • the walkway, lobby or basement connecting them does not contain a residential unit
  • or, the wall connecting them does not contain any normal use doors

Then you need to regiser each structure as its own building.

If there are two or more structures connected together by either:

  • a walkway, lobby or basement containing a residential unit
  • a wall with a normal use door

Then it is not a simple building.

If your building contains multiple structures with residential units, then it is not a simple building.

Mixed use buildings can be simple buildings.

For example, The Gherkin is a simple high-rise building because it is a single structure not connected to any other structures.
