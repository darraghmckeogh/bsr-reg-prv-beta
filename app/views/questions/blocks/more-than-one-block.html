Count the number of sections in {{ data['building-name'] }}

A section is a collection of units in the building that share a main entrance, or can only be accessed by crossing a party wall, walkway, shared lobby or basement.

Count each independent structure, like a tower block.

Count each collection of residential units that can be accessed from the same main entrance or main entrances. For example, floors in the building that can only be accessed by their own lift or staircase.

Count each collection of residential units that can only be accessed by crossing a party wall, walkway, shared lobby or basement.

Do not count each residential unit as its own section.

Do not count collections of non residential units.

For example, a simple tower, in which all of the flats can be accessed from the same main entrances is not made up of multiple sections.

Two tower blocks that share a basement containing one or more residential units would be made up of two sections.
